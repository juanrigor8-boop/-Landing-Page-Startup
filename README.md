# 🚀 DevNotes - Modern Startup Landing Page

¡Bienvenido a DevNotes! Este es un proyecto de práctica de desarrollo web enfocado en crear una interfaz de usuario de alto impacto visual, utilizando una estética **Dark Neon** y técnicas modernas de CSS.

El objetivo principal es demostrar habilidades en la creación de layouts comerciales, microinteracciones y animaciones fluidas.

---

## ✨ Características Principales

* **Diseño de Vanguardia:** Interfaz en modo oscuro con acentos neón cian y púrpura.
* **Efecto Glassmorphism:** Paneles con desenfoque de fondo y bordes translúcidos.
* **Animaciones Avanzadas:** * `@keyframes` para el efecto flotante del Hero.
    * Revelado suave de texto al cargar la página.
    * Sistemas de partículas visuales simulados con sombras.
* **Microinteracciones:** Botones con efectos de pulsación y tarjetas reactivas al movimiento del mouse.
* **Diseño Responsivo:** Adaptable a diferentes tamaños de pantalla (Mobile First).

---

## 🛠️ Tecnologías Utilizadas

| Tecnología | Uso |
| :--- | :--- |
| **HTML5** | Estructura semántica de la landing page. |
| **CSS3** | Estilizado avanzado, Flexbox, Grid y Animaciones. |
| **Google Fonts** | Tipografía 'Inter' para un look limpio y moderno. |

---

## 💡 Aprendizajes Clave

Durante el desarrollo de este proyecto, se pusieron en práctica los siguientes conceptos:

* **Jerarquía Visual:** Uso de gradientes lineales para dirigir la atención del usuario.
* **Cubic-Bezier:** Curvas de tiempo personalizadas para que las transiciones no sean lineales y se sientan más "orgánicas".
* **Box-Shadows Compuestos:** Capas de sombras para crear el resplandor de neón sin sacrificar el rendimiento.
